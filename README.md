## COZY VIBECODED SNAKE

### Description of the Functionalities:
In here you will find a cozy vibecoded Snake game, written in HTML, CSS and JavaScript. Start the program by switching into the "snake" folder and open the file "index.html" in a local webbrowser of your choice on your device. After this, the Snake game should be ready for play. You can control the game's character with the arrow keys of your keyboard. Completely vibecoded with ChatGPT.
